esci: effect sizes and confidence intervals
esci is free software to better understand and analyze data. There are two components:
esci for data analysis, available in R, jamovi, and JASP, maintained by Robert Calin-Jageman
  • for data analysis and great graphs with CIs
esci web in JavaScript, by Gordon Moore
  • for statistical dances and interactive graphical tools
